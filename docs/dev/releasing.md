# Releasing

Not yet defined — see [Phase 6 in the roadmap](../phases/index.md).

The first usable release is **v0.1**, which lands at the end of Phase 6
(Producer + lifecycle). Until then there are no tagged versions and no
PyPI uploads. The package metadata is in place
(`pyproject.toml` → `[project]`) but the build is not exercised in CI as a
release artifact yet.

When release tooling lands, it will be documented here.
